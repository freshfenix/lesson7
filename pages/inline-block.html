Inline-Block — демонстрація

1. Inline vs Inline-Block vs Block

inline (span) inline (a) inline-block (span)
inline>inline-block C
block 1 (div)
block 2>display:inline-block (div)

Порада: змінюйте display у inline elements, щоб побачити різницю; inline не приймає width/height, inline-block — приймає.

2. Пробіли між inline-block елементами (whitespace)

A B C

Ці елементи розділені пробілами в HTML. Щоб прибрати проміжок — можна видалити пробіли або використовувати font-size:0 на контейнері.

3. Vertical-align

baseline top middle bottom

vertical-align працює з inline/inline-block/table-cell; експериментуйте зі значеннями.

4. Короткі пояснення та код

<span style="display:inline-block">inline-block</span>
<a style="display:block; width:100%">block</a>
/* Щоб прибрати whitespace між inline-block елементами: font-size:0 на контейнері */
